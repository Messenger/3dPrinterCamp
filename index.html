3dPrinterCamp

A RepRap Mendel90 variant used for helping people build their own printer in one day.

Download this project as a .zip file Download this project as a tar.gz file

The goal of this build is to expand the functionality of the printer and to make it easier to assemble using SAE parts sourced in the United States.
